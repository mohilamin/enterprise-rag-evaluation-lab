I built an Enterprise Document Intelligence + RAG Evaluation Lab as a portfolio project.

The goal was to go beyond a basic "chat with PDF" demo. Enterprise RAG needs measurable retrieval quality, citations, stale-document detection, sensitive-data controls, hallucination-risk scoring, and repeatable evaluation evidence.

What I built:

- Synthetic enterprise document corpus
- Golden question set
- Document ingestion and normalization
- Chunking with metadata
- TF-IDF retrieval baseline
- Citation-grounded deterministic answer service
- Hit@K and MRR retrieval evaluation
- Groundedness, citation coverage, stale-risk, and hallucination-risk scoring
- FastAPI endpoints
- Streamlit dashboard
- pytest, Ruff, Docker, and GitHub Actions

Tech stack:
Python 3.12, pandas, scikit-learn, DuckDB, FastAPI, Streamlit, pytest, Ruff, Docker.
