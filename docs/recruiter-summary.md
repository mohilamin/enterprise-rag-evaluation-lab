# Recruiter Summary

This project shows AI Data Engineering and MLOps skills through a realistic enterprise RAG evaluation system.

## What It Demonstrates

- Synthetic enterprise document generation.
- Document ingestion and metadata validation.
- Chunking and retrieval index creation.
- Citation-grounded answer composition.
- RAG evaluation metrics.
- FastAPI and Streamlit delivery.
- Tests, linting, Docker, and CI.

## Why It Matters

Companies need trustworthy AI assistants over internal knowledge. This project shows how to evaluate retrieval quality and answer safety before depending on a GenAI system.

## Role Fit

- Data Engineer: ingestion, transformation, structured outputs.
- AI Data Engineer: document preparation and RAG readiness.
- MLOps Engineer: evaluation, scoring, reproducibility.
- Data Platform Engineer: API, dashboard, CI, Docker, and architecture.
